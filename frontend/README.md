# Frontend

## Requisitos Previos

- [NodeJS](https://nodejs.org/en) (Versión X.X)

## Configuración de Entorno

### 1. Accede a la carpeta "frontend" dentro del repositorio clonado:

```
cd frontend
```

### 2. Instalar las dependencias

```
npm install
```

### 3. Ejecutar la pagina

```
npm run dev
```
